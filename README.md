# Hv2ray
A Linux GUI tool for v2ray based on Qt.

## Dependencies
- Qt >= 4
- python3

## Compile
```bash
git clone https://github.com/aliyuchang33/Hv2ray && cd Hv2ray
mkdir build && cd ./build
qmake ../
make
```

## Run

```bash
./Hv2ray
```
