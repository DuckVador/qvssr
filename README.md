## Dependencies
- Qt >= 5
- python3.5 | 3.6 | 3.7

## Compile
```bash
git clone https://github.com/lhy0403/Hv2ray && cd Hv2ray
mkdir build && cd build
qmake ./
make
```

## Run

```bash
./Hv2ray
```
