# Hv2ray
An linux GUI tool for v2ray based on Qt.

## Dependencies
- Qt >= 4
- python3 (Required if you need `vmess2json.py`)

## Compile
```
git clone https://github.com/aliyuchang33/Hv2ray && cd Hv2ray
mkdir build && cd ./build
qmake ../
make
```
